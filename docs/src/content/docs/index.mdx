## Quick Start

**macOS** — no Rust toolchain needed:

```bash
brew install cmakefmt/cmakefmt/cmakefmt
```

**Any platform** — if you already have Rust:

```bash
cargo install cmakefmt-rust
```

**Linux, macOS, and Windows** — download the pre-built binary from [GitHub Releases](https://github.com/cmakefmt/cmakefmt/releases/latest).

Then:

```bash
cmakefmt --version
cmakefmt --dump-config > .cmakefmt.yaml   # generate a starter config
cmakefmt --check .                        # dry-run: show what would change
cmakefmt --in-place .                     # apply formatting
cmakefmt --staged --check                 # pre-commit: format only staged files
```

## Performance

**Geometric-mean speedup: `22.18×`** across the full corpus.
On a 612-file real-world repo ([oomph-lib](https://github.com/oomph-lib/oomph-lib)): `~97×` faster than `cmake-format` sequentially, and `~240×` faster with `--parallel 8`.
