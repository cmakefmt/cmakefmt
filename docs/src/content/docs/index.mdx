## Quick Start

<InstallCmd label="macOS" cmd="brew install cmakefmt/cmakefmt/cmakefmt" />
<InstallCmd label="Cargo" cmd="cargo install cmakefmt-rust" />
<InstallCmd label="pip" cmd="pip install cmakefmt" />

Or download a pre-built binary from [GitHub Releases](https://github.com/cmakefmt/cmakefmt/releases/latest).

Then:

```bash
cmakefmt --version
cmakefmt init                             # generate a starter config
cmakefmt --check .                        # dry-run: show what would change
cmakefmt --in-place .                     # apply formatting
cmakefmt --staged --check                 # pre-commit: format only staged files
```
