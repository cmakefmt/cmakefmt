This page lists open-source projects that format their CMake files with
`cmakefmt`. Want your project listed here? [Open a pull request](https://github.com/cmakefmt/cmakefmt/edit/main/docs/src/content/docs/users.md)
and add it to the table below.

## Projects

| Project | Description | Config |
|---------|-------------|--------|
| [cmakefmt](https://github.com/cmakefmt/cmakefmt) | The formatter itself — dogfoods its own formatting | Default |

## Add your project

To add your project, open a pull request that appends a row to the table
above. Include:

- **Project** — a Markdown link to the repository or homepage
- **Description** — one sentence describing the project
- **Config** — `Default`, or a brief note if you use a custom config (e.g.
  `line_width = 100`)
