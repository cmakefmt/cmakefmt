## Add your project

Using `cmakefmt` in your project? [Open a pull request](https://github.com/cmakefmt/cmakefmt/edit/main/docs/src/content/docs/users.md)
or [let us know on GitHub Discussions](https://github.com/cmakefmt/cmakefmt/discussions)
and we'll add you here. Include:

- **Project** — a link to the repository or homepage
- **Description** — one sentence describing the project
- **Config** — `Default`, or a brief note if you use a custom config (e.g.
  `line_width = 100`)
